# 修改日志 - 2026-07-13 - 入库单 Header 增加金额与物料数

## 变更原因

入库单 Header 需要承担单据摘要能力，用户不展开也应能看到本次入库总金额和涉及物料项数。根据列表与详情字段一致规则，展开明细底部也同步展示入库总金额，确保 Header 摘要和明细汇总口径一致。

## 用户可见变化

- 入库单 Header 新增 `入库总金额`。
- 入库单 Header 新增 `N 项物料`。
- 入库单展开后的 `入库物料` 底部同步展示 `入库总金额`。
- 不新增总数量汇总，避免不同单位混合后产生误解。

## PRD / Mock 数据更新

- Console 库存管理 PRD 已同步入库单 Header 字段与展开明细底部汇总规则。
- Mock 数据未调整；入库总金额由现有入库物料明细金额求和。
