## 修改原因

库存流水列表中的 `数量变化 / 结存数量` 字段标题过长，且原先用文字标签展示 `库存上升` / `库存下降`，视觉上和数量不够对齐。调整为更简洁的字段名和方向 icon 展示。

## 用户可见变化

- 库存流水列表字段名由 `数量变化 / 结存数量` 改为 `库存变化`。
- 物料详情页的库存流水明细同步使用 `库存变化` 字段名，避免同一模块口径不一致。
- 库存上升 / 下降不再使用文字标签展示。
- 库存变化数字右侧使用固定尺寸方向 icon 表示上升或下降。
- 图标改为纯 icon 展示，通过 `inline-flex`、固定宽高、SVG block 和居中对齐，避免图标相对数字歪斜。
- 库存变化数字与方向 icon 改为固定两列布局：数量右对齐，方向 icon 按列对齐，避免因数量长度不同造成 icon 横向跳动。
- 修正固定两列布局导致的上下错位问题：`结存` 文案现在也纳入同一个网格并右对齐，避免短数量如 `+2个` 被推到右侧后与下方结存信息脱节。
- 删除库存流水标题下方的说明文案 `整体查询历史库存变化，列表中用方向图标展示库存上升或下降`，减少页面重复解释。
