# 修改日志 - 2026-07-07 - 当前库存单单位展示

## 变更原因

入库流程已经删除入库换算配置，库存管理首页和盘点页的 `当前库存` 不应继续展示包装约数，避免用户误以为系统仍按入库换算维护双单位库存。

## 用户可见变化

- Console 库存管理首页物料列表的 `当前库存` 只展示数量 + 核算单位，例如 `14000kg`。
- Console 库存盘点页的 `当前库存` 只展示数量 + 核算单位；包装信息仍由 `规格` 列承担。
- 物料存在未处理盘点差异时，`含未处理差异` 标签保留。
- 物料详情页的未处理盘点差异提示恢复可用，按该物料待处理差异汇总展示差异数量。

## PRD / Mock 数据更新

- Console 库存管理 PRD 已同步 `当前库存` 单单位展示口径。
- 未新增 mock 数据；现有库存汇总数据已能覆盖饲料、药品、消耗品和工具展示。
