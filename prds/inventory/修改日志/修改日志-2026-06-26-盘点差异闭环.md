# 修改日志 - 2026-06-26 - 盘点差异闭环（库存口径）

## 变更日期

2026-06-26

## 背景与原因

原盘点在「盘点差异确认」点击 `确认并调整库存` 时会立即修改库存：盘亏按 FEFO 扣减、盘盈生成盘盈调整批次，用户没有机会判断差异来源，盘盈/盘亏口径粗糙且容易污染业务消耗。本次按与用户对齐的方案，把盘点改为「提交只产差异、逐条处理才改库存」的闭环，先完成库存口径，财务模块后续再做。

## 受影响文件

- `src/components/inventory/inventoryData.ts`
  - `InventoryLedgerType` 新增 `消耗冲销`、`入库更正`。
  - 新增差异相关类型：`InventoryDifferenceDirection`、`InventoryDifferenceStatus`、`InventoryDifferenceFinanceStatus`、`InventoryDifferenceMethod`、`InventoryDifferenceMethodMeta`、`InventoryDifferenceRecord` 及处理/提交入参出参类型。
  - 新增 `inventoryDifferenceMethodMetas`（8 种处理方式 → 库存流水类型、批次落点、消耗影响、财务状态的映射）。
  - 新增 `buildInventoryStocktakeDifferences`（提交只产差异、不改库存）与 `buildInventoryDifferenceResolution`（按处理方式对当前库存做差量调整、落批次与成本、产出库存流水与财务状态），含 `deductInventoryQuantity`、`resolveMaterialStockBase` 辅助函数。
  - 新增 `inventorySeedDifferences` 待处理差异 mock（氟苯尼考 +50ml、妊娠母猪料 -40kg、连续注射器 +2个）。
- `src/components/inventory/ConsoleInventoryPage.tsx`
  - 提交盘点改为生成待处理差异、盘点单置 `待确认`，不再直接改库存；完成页改为「盘点已提交 / 去处理差异」。
  - 新增 `difference-list` 视图（差异列表，含分类/方向/处理状态/财务状态筛选、差异数量排序）与差异处理抽屉（来源单选、关联批次选择、影响预览、财务待确认提示）。
  - 首页右上角新增 `盘点差异` 按钮（角标显示待处理条数）+ 首页提示条；物料详情页新增未处理差异提示条；库存列表新增 `含未处理差异` 标签。
  - 库存流水筛选项新增 `消耗冲销`、`入库更正`；消耗趋势/Top 物料统计将 `消耗冲销` 按负向计入，修复冲销被算成消耗的问题。
  - 执行/确认/详情页统一中性口径 `实盘多出` / `实盘少了`。
- `src/styles.css`
  - 新增差异列表、差异处理抽屉、影响预览、处理方式单选卡、未处理差异标签、差异提示条等样式。
- `src/components/inventory/inventoryUtils.test.ts`
  - 调整盘盈调整批次断言指向数据层；新增差异闭环断言（盘点差异处理、difference-list、两个 builder、新增流水类型、不再保留「确认并调整库存」）。
- `Inventory management/Inventory PRD/01-Console库存管理PRD.md`
  - 重写盘点差异确认 / 完成 / 库存流水联动；新增 `8.11 盘点差异处理`（入口、差异列表、处理抽屉、方向≠结论、处理方式映射表、外显、时序快照）；过期报废顺延为 `8.12`；库存流水类型表补充 `消耗冲销`、`入库更正`。

## 用户可见变化

- 提交盘点不再立即改库存，而是生成「待处理差异」。
- 首页右上角新增带角标的 `盘点差异` 入口；存在待处理差异时首页与物料详情有提示条，库存列表有 `含未处理差异` 标签。
- 新增「盘点差异处理」页：列表 + 处理抽屉，可按差异方向选择 8 种来源，确认后库存与批次才更新。
- 录入/确认/列表统一使用 `实盘多出` / `实盘少了`；`盘盈` / `盘亏` 仅保留给「原因不明」结论。
- 涉及采购金额的处理（入库少录、赠品、原因不明盘盈/盘亏）标记 `财务待确认`，本期不算钱。
