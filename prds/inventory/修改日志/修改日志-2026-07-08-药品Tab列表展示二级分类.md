# 修改日志 - 药品 Tab 列表展示二级分类

- **日期**：2026-07-08
- **影响文件**：
  - `src/components/inventory/inventoryData.ts`
  - `src/components/inventory/ConsoleInventoryPage.tsx`
  - `src/components/inventory/inventoryUtils.test.ts`

## 变更原因

药品 Tab 下用户需要在列表直接区分疫苗、兽药、保健品，不能只在 Tab 层级看到「药品」而无法判断具体类型。

## 用户可见变化

- 库存首页切换到「药品」Tab 时，列表新增「药品分类」列，展示疫苗 / 兽药 / 保健品。
- 该列支持筛选与排序；其他 Tab 不展示此列（避免与 Tab 分类重复）。

## 验证

- `npm run build`
- `npx tsx src/components/inventory/inventoryUtils.test.ts`
