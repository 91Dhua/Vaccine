# 修改日志 - 2026-07-13 - 物料详情基础信息补齐

## 变更原因

物料详情页 `基础信息` 未完整对应添加物料时采集的档案字段，尤其缺少入库快建时维护的 `单位换算`，导致用户无法在详情页核对物料档案与入库单位选择依据。

## 用户可见变化

- 物料详情 `基础信息` 增加 `单位换算`。
- 物料详情 `基础信息` 增加 `物料名称(英文)`、`品牌名称(英文)`。
- 物料详情通用字段文案调整为与添加物料表单一致：`物料名称(中文)`、`品牌名称(中文)`、`物料类型`。
- `单位换算` 多条规则用 `；` 分隔展示；未维护换算时展示 `—`。

## PRD / Mock 数据更新

- Console 库存管理 PRD 已同步物料详情基础信息字段，要求与添加 / 编辑物料时采集的档案字段逐项对应。
- Mock 数据未新增；现有种子物料已包含英文名称、英文品牌和单位换算，可直接验证展示。
