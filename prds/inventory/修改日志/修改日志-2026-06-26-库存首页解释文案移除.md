# 修改日志-2026-06-26-库存首页解释文案移除

## 变更日期

2026-06-26

## 受影响文件

- `src/components/inventory/ConsoleInventoryPage.tsx`
- `src/components/inventory/inventoryUtils.test.ts`
- `Inventory management/Inventory PRD/01-Console库存管理PRD.md`
- `Vaccine/prds/inventory/修改日志/修改日志-2026-06-26-库存首页解释文案移除.md`

## 变更原因

库存首页经营分析卡片标题下存在多句解释性副文案，包括资金占用说明、Top 物料说明、风险优先级说明和趋势价值说明。该类文案对当前首页操作帮助有限，且增加了视觉噪音。

## 用户可见变化

- `库存金额占比` 卡片不再展示 `统一金额口径，方便场长快速看清库存资金占用`。
- `消耗金额 Top 物料` 卡片不再展示 `本月 · 全部分类 · 主要花费在哪些物资上`。
- `库存风险` 卡片不再展示 `按风险优先级集中查看待处理物料` 或筛选态的风险列表说明。
- `库存消耗趋势` 卡片不再展示 `按日查看消耗金额波动，比单个汇总数字更适合做经营判断`。

## PRD / Mock 数据更新

- Console 库存管理 PRD 已补充经营分析卡片标题下不展示解释性副文案的规则。
- 本次仅移除展示文案，不涉及业务字段、计算逻辑或 mock 数据更新。

## 验证

- `npm run build`：通过。
- `npx tsx src/components/inventory/inventoryUtils.test.ts`：通过（退出码 0）。
- 已检索 `ConsoleInventoryPage.tsx` 与 `Inventory management/Inventory PRD/01-Console库存管理PRD.md`，确认被移除的解释性文案无残留。
