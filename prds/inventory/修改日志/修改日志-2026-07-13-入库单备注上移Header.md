## 变更原因

入库单 Header 已经展示入库单号、入库类型、入库时间和经办人，展开后的 `基本信息` 模块与 Header 高度重合，增加了用户扫表成本。备注是单据摘要信息，应上移到 Header，长备注用省略展示并通过 Hover 查看完整内容。

## 用户可见变化

- 入库单 Header 新增 `备注` 展示。
- 备注默认单行展示约 18 个中文字宽，超出后省略；鼠标 Hover 可查看完整备注。
- 入库单展开内容删除 `基本信息` 模块，仅保留 `入库物料` 明细。

## PRD / Mock 数据更新

- Console 库存管理 PRD 已同步入库单 Header 与展开内容展示规则。
- Mock 数据未调整；继续复用现有入库单备注数据。

## 验证

- 已执行 `npx tsx src/components/inventory/inventoryUtils.test.ts`。
- 已执行 `npm run build`。
- 已通过本地页面 `http://localhost:5173/` 验证：入库单 Header 展示 `备注：7月份采购`，备注文本使用省略样式；展开区仅展示 `入库物料`，不再展示 `基本信息`。
