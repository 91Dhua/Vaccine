# 修改日志 - 2026-06-26 - 库存消耗趋势图重做

## 变更日期

2026-06-26

## 背景与原因

库存首页原 `本月消耗趋势` 图表使用 `preserveAspectRatio="none"` 的 SVG，线条与数据点会随容器宽度被拉伸变形，且缺少 Y 轴金额刻度、坐标对齐和交互，用户反馈“UI 很差、压根不想看”。本次按业内主流经营看板标准重做该图表，并补全时间范围与分类筛选。

## 受影响文件

- `src/components/inventory/ConsoleInventoryPage.tsx`
  - 新增 `InventoryAnalyticsTimeRange` 类型与 `inventoryAnalyticsTimeRangeOptions` 常量；分类筛选 `inventoryAnalyticsCategoryFilterOptions` 扩展为 `全部 / 饲料 / 兽药 / 疫苗 / 消毒用品 / 保健品 / 工具`。
  - 移除旧的 `buildTrendChartCoordinates`、`buildVisibleTrendWindow` 及拉伸式 SVG 渲染。
  - 新增 `InventoryTrendChart` 组件：用 `ResizeObserver` 测量容器宽度后按真实像素绘制，保证线条/圆点不变形；包含 Y 轴金额刻度、X 轴日期刻度、网格基线、面积渐变、平滑折线（Catmull-Rom 转贝塞尔）、峰值高亮点，以及鼠标吸附最近一天的参考线 + 浮层提示；无数据时显示空态文案。
  - 新增 `analyticsTimeRange`（默认 `本月`）与 `analyticsCustomRange` 状态；新增 `analyticsDateWindow` 计算时间窗口（近7天 / 近30天 / 本月 / 上月 / 自定义）。
  - 趋势数据与 `消耗金额 Top 物料` 改为按所选时间窗口 + 分类联动计算（`consumptionWindowLedgers`、`consumptionTrendPoints`、`windowTopSpendRows` 等），概览指标改为区间累计 / 日均 / 峰值日 / 消耗天数。
  - 图表更名为 `库存消耗趋势`；删除不再使用的 `currentMonthConsumptionCost`。
- `src/styles.css`
  - 移除旧趋势图（grid / bars / column / bar-shell / label / 拉伸式 svg）样式。
  - 新增测量式折线图样式：`__svg`、`__gridline`、`__ytick`、`__xtick`、`__area`、`__line`、`__dot(.is-peak)`、`__cursor`、`__empty`、`__tooltip`，以及时间范围/分类筛选布局 `inventory-trend-filters`、`__head--trend`。
  - 更新响应式断点中相关规则。
- `src/components/inventory/inventoryUtils.test.ts`
  - 断言更名为 `库存消耗趋势`、不再保留 `本月消耗趋势`。
  - 断言时间范围选项 `["近7天", "近30天", "本月", "上月", "自定义"]`、默认 `本月`、分类选项扩展到 `保健品 / 工具`。
- `Inventory management/Inventory PRD/01-Console库存管理PRD.md`
  - 更新图表 H 说明：更名、时间范围筛选、分类筛选扩展、交互与联动指标、空态文案。

## 用户可见变化

- 图表名称由 `本月消耗趋势` 改为 `库存消耗趋势`。
- 新增时间范围筛选 `近7天 / 近30天 / 本月 / 上月 / 自定义`，默认 `本月`；`自定义` 出现日期区间选择器。
- 分类筛选扩展为 `全部 / 饲料 / 兽药 / 疫苗 / 消毒用品 / 保健品 / 工具`。
- 折线图不再变形，新增 Y 轴金额刻度、X 轴日期刻度、网格基线、面积渐变、峰值高亮点。
- 鼠标在图表上移动可吸附到最近一天，展示参考线与日期/金额浮层。
- 概览指标随所选区间联动：累计消耗、日均消耗、峰值日、消耗天数；`消耗金额 Top 物料` 同步按区间 + 分类联动。
- 所选区间无数据时图表与 Top 物料均显示空态文案。

## Mock 数据

- 复用现有 6 月业务消耗流水（6/17–6/26，覆盖饲料 / 兽药 / 疫苗 / 消毒用品），默认 `本月` 视图即可看到完整折线与峰值；`保健品`、`工具` 暂无消耗流水，用于验证空态。
