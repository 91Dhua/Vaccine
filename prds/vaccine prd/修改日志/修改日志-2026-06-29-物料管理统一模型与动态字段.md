# 修改日志：物料管理接入统一物料模型与分类动态字段

## 变更日期

2026-06-29

## affected files

- `src/components/VaccineCatalogPage.tsx`
- `src/components/inventory/inventoryData.ts`
- `src/components/inventory/MaterialProfileFields.tsx`（新增）
- `src/styles.css`
- `prds/vaccine prd/01-疫苗管理.md`

## 变更原因

物料管理原使用独立的疫苗目录模型（中/英文名、中/英文品牌 + 仅疫苗专属字段），与库存物料模型不一致，且非疫苗分类缺少专属字段。本次将物料管理接入与库存共用的统一物料模型，并按分类动态展示专属字段。

## 用户可见变化

- 物料管理列表与库存共用统一物料模型，列表新增 `核算单位` 列，物料名称旁展示 `资料待完善` 标记。
- 分类 tab 显示各分类物料数量。
- 单条添加由两步弹窗改为单页弹窗：通用档案字段（名称中/英、分类、品牌中/英、核算单位、安全库存、备注）+ 选择分类后动态展示的专属字段。
- 各分类专属字段：兽药（剂型/使用方式/休药期）、疫苗（疫苗类型/接种方式/剂型/是否冷链/储存温度/单次剂量/免疫有效期/休药期/免疫间隔期）、饲料（形态/适用猪只/适用阶段）、消毒用品（有效成分/稀释比例/适用场景）、保健品（主要成分/使用方式）。
- 饲料专属字段后续已在 2026-06-30 调整为 `适用猪只与阶段`：一行选择猪只类型并多选阶段，可添加多行，不再填写起始/结束天数区间。
- 批量添加模版改为通用档案列（物料类型/物料名称中英/品牌中英/核算单位/安全库存）；专属字段不在模版内，缺必填专属项的导入物料标记 `资料待完善`。
- 物料详情按分类展示通用 + 专属字段。

## PRD / mock data 更新
