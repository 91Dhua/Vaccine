# 修改日志：物料管理批量添加

## 变更日期

2026-06-29

##  affected files

- `src/App.tsx`
- `src/components/VaccineCatalogPage.tsx`
- `src/styles.css`
- `package.json`
- `package-lock.json`
- `prds/vaccine prd/01-疫苗管理.md`

## 变更原因

设置页原入口与页面标题为药品管理，无法覆盖饲料、工具等非药品物料，也缺少按物料分类查看和批量初始化录入能力。本次按物料管理口径统一页面名称，并补充分类 tab 与 Excel 批量添加入口。

## 用户可见变化

- 设置菜单入口由 `药品管理` 调整为 `物料管理`。
- 主页面标题调整为 `物料管理`，说明文案覆盖饲料、兽药、疫苗、消毒用品、保健品、工具、其他。
- 主页面增加分类 tab：全部、饲料、兽药、疫苗、消毒用品、保健品、工具、其他，并显示各分类数量。
- 原 `添加药品` 按钮调整为 `添加`。
- `添加` 按钮左侧新增 `批量添加`，支持下载 Excel 模版并上传填写后的文件完成物料批量添加。
- 表格、详情、编辑、添加表单中的类型文案由药品类型调整为物料类型。

## PRD / mock data 更新

- 更新 `prds/vaccine prd/01-疫苗管理.md`，将页面产品口径调整为物料管理，并补充分类 tab、单条添加、批量添加和异常情况说明。
- mock 数据补充饲料、工具、其他分类样例，便于直接验证分类 tab。
- 使用浏览器原生能力生成 Excel 2003 XML `.xls` 模版并读取上传后的模版表格，未新增第三方 Excel 依赖。
