# 修改日志：Console 任务配置说明卡删除

## 变更日期

2026-06-19

## 影响文件

- `src/components/MedicalTaskSchemePage.tsx`
- `src/styles.css`
- `prds/medical-task/00-疫苗任务与治疗任务统一入口方案.md`

## 变更原因

任务配置页中原有的 `任务配置` 说明卡仅展示通用提示和任务类型标签，缺少实际业务信息。放在创建对象摘要和处方药物表单之间会打断用户填写流程，也让页面显得冗余。

## 用户可见变化

- 任务配置页删除独立的 `任务配置` 说明卡。
- 用户进入配置页后，从创建对象摘要直接进入处方药物/任务表单。
- 疫苗、兽药、保健品配置页仍保持一致页面骨架，但不再额外展示空泛说明卡片。

## PRD / Mock Data 更新

- PRD 更新任务配置页面结构，移除 `任务配置标题区`。
- Mock 数据无变化。

## 验证执行

- `npm run build`
- 浏览器验证 `http://127.0.0.1:5173/` Console 治疗「先选任务类型」：
  - 进入治疗任务配置页后不再展示 `任务配置` 说明卡。
  - 创建对象摘要后直接展示 `处方药物` 和药物表单。
